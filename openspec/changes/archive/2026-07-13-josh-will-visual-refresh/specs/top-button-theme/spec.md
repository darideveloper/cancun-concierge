## ADDED Requirements

### Requirement: TOP button uses dark beige color

The "TOP" scroll-to-top button in the footer and its arrow SVG icon SHALL use `#c9ad85` to match the updated color palette.

#### Scenario: TOP button text color

- **WHEN** the `a.btn.footer` element with text "TOP" renders within `#footer.josh-will`
- **THEN** its color SHALL be `#c9ad85`

#### Scenario: TOP button arrow icon color

- **WHEN** the SVG arrow icon inside the TOP button renders
- **THEN** its fill SHALL be `#c9ad85`
